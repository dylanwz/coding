# Standard Tags
The standard patterns are listed below:

- `BT`:     backtracking,
- `DP`:     dynamic programming,
- `HEAP`:   heap,
- `FLOOD`:  flood-fill algorithms (on top of graphs/DFS)
- `DIFF`:   difference arrays,
- `HASH`:   using a hash table to reduce linear operations to constant time. 
- ... .

Prefix by "`TAG-`".
